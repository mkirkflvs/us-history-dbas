HOW THEY WORK:
=================================================================

Each DBA file contains:
- Topic cards organized by standard
- Expandable panels with historical context
- Images from authoritative sources (Britannica, Met Museum, PBS, etc.)
- FALLBACK SYSTEM: If an image doesn't load, a clickable link appears instead

IMAGES MIGHT NOT LOAD IF:
1. Your school network blocks certain domains
2. The source website has access restrictions
3. Browser/network bandwidth limitations

WHEN IMAGES DON'T LOAD:
- You'll see a "Click to open image" link instead
- Click the link to open the image in a new tab
- Full caption tells you what the image is

IF YOU NEED TO FIX IMAGE URLS:
=================================================================

Each image has:
1. <img src="URL"> - the image source
2. <img alt=""> - fallback text
3. <a href="URL"> - clickable link if image fails
4. <div class="img-cap"> - image caption

To replace an image URL:
1. Find the image ID (like "causes-cw", "dday", etc.)
2. Find its <img src="URL">
3. Replace the URL with a working image link
4. Update the caption if needed
